# R-Hub Enterprise Windows Infrastructure Laboratory

## Overview
R-Hub is a self-built enterprise Windows infrastructure project designed to simulate a real-world IT operations environment.

## Technologies
- Windows Server 2022
- Active Directory (AD DS)
- DNS
- Group Policy (GPO)
- Organizational Units (OU)
- Domain Join
- PowerShell
- VMware Workstation
- GLPI (IT Service Management)

## Project Highlights
- Built an enterprise Active Directory environment.
- Configured DNS, OUs, Group Policy, and Domain Join.
- Implemented GLPI for IT Service Management.
- Automated administrative tasks using PowerShell.
- Documented the complete deployment process.
